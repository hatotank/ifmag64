## Overview

Susie 64-bit plugin for the MAG image format (personal use).

## Build

Run `build.bat` from the repository root in a Command Prompt. The `ifmag64.sph` file will be created in the `dist` folder.  
Note: This project is prepared for Visual Studio 2010 (x64 tools). Edit `build.bat` as necessary for your environment.

## Notes

- This plugin was developed to follow TORO's "Susie 32bit / 64bit Plug-in" specification (2026-02-11) as closely as possible: https://toro.d.dooo.jp/dlsphapi.html  
- Intended for personal use; thorough testing has not been performed.
